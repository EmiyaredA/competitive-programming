在 Python 中，`set` 是一个**无序、元素唯一、可变**的集合类型，非常适合用于去重、集合运算（交并差）等场景。下面我系统讲一下 `set` 的常用用法👇

---

## 🌱 一、创建 set

```python
# 直接用大括号创建
s = {1, 2, 3}

# 用 set() 函数创建（推荐）
s2 = set([1, 2, 2, 3])   # 自动去重
print(s2)  # 输出 {1, 2, 3}

# 创建空集合
s3 = set()
```

> ⚠️ 注意：`{}` 是空字典，不是空集合！要用 `set()` 创建空集合。

---

## 🔹 二、常用操作

### 1️⃣ 添加元素

```python
s = {1, 2, 3}
s.add(4)           # 添加单个元素
s.update([5, 6])   # 添加多个元素（可迭代对象）
```

### 2️⃣ 删除元素

```python
s = {1, 2, 3}
s.remove(2)   # 删除元素，如果不存在会报错
s.discard(4)  # 删除元素，不存在也不会报错
s.pop()       # 随机删除一个元素（因为 set 无序）
s.clear()     # 清空集合
```

---

## 🔹 三、集合运算

Python 的 `set` 自带非常方便的数学运算符👇

| 运算  | 方法                              | 说明               | 示例                                   |        |                  |
| --- | ------------------------------- | ---------------- | ------------------------------------ | ------ | ---------------- |
| 交集  | `&` 或 `.intersection()`         | 两集合共有的元素         | `{1,2,3} & {2,3,4}` → `{2,3}`        |        |                  |
| 并集  | `                               | `或`.union()`     | 所有元素（自动去重）                           | `{1,2} | {2,3}`→`{1,2,3}` |
| 差集  | `-` 或 `.difference()`           | 属于A但不属于B的元素      | `{1,2,3} - {2}` → `{1,3}`            |        |                  |
| 对称差 | `^` 或 `.symmetric_difference()` | 属于A或B但不同时属于两者的元素 | `{1,2,3} ^ {2,3,4}` → `{1,4}`        |        |                  |
| 子集  | `.issubset()`                   | 判断 A 是否是 B 的子集   | `{1,2}.issubset({1,2,3})` → `True`   |        |                  |
| 超集  | `.issuperset()`                 | 判断 A 是否包含 B      | `{1,2,3}.issuperset({1,2})` → `True` |        |                  |
| 不相交 | `.isdisjoint()`                 | 判断两集合是否无公共元素     | `{1,2}.isdisjoint({3,4})` → `True`   |        |                  |

## 🔹 四、遍历与基本操作

```python
s = {1, 2, 3}
for x in s:
    print(x)

print(len(s))   # 元素个数
print(2 in s)   # 判断元素是否在集合中
```

---

## 🔹 五、集合去重示例

```python
nums = [1, 2, 2, 3, 3, 4]
unique_nums = list(set(nums))
print(unique_nums)  # [1, 2, 3, 4]（顺序可能不同）
```

---

## 🔹 六、冻结集合（不可变）

```python
fs = frozenset([1, 2, 3])
# fs.add(4) 会报错，因为 frozenset 是不可变的
```
